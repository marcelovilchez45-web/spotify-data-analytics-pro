## 📌 Descripción del Proyecto
Este proyecto aborda el desafío de analizar grandes volúmenes de datos en la industria musical. Desarrollé un flujo de datos completo (**ETL**) que transforma registros crudos de Spotify en un ecosistema de Inteligencia de Negocios, permitiendo identificar patrones de éxito, medir la fidelización de fans y predecir tendencias de mercado.

---

## 📐 Arquitectura Técnica

1.  **Data Foundation:** Limpieza, normalización y tratamiento de **+8,500 registros** en MySQL, asegurando la integridad referencial y eliminando redundancias.
2.  **Modeling:** Diseño de un **Esquema en Estrella (Star Schema)** optimizado para analítica, vinculando tablas de dimensiones (Artistas, Álbumes, Tracks) con métricas de rendimiento.
3.  **Analytics Layer:** Implementación de **Vistas SQL** avanzadas para pre-procesar cálculos complejos, garantizando que el reporte en Power BI sea fluido y eficiente.

## 🚀 Impacto y Valor de Negocio
* **Fidelización Estratégica:** Capacidad para diferenciar artistas de impacto momentáneo de aquellos con una base de fans sólida para inversiones a largo plazo.
* **Optimización de Producto:** Identificación de la "ventana de éxito" (3.0 - 3.5 min) para la duración de canciones en el mercado actual.
* **Gobierno de Datos:** Una estructura escalable que permite añadir nuevos años de datos sin comprometer el rendimiento del reporte.
